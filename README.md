# Análise Semântica de Detratores - E-commerce Olist

Um projeto de analise de dados focado em analisar o sentimento dos clientes que comentaram\fizerem review com apenas uma estrela em produtos comprados digitalmente, levando em conta dados reais retirados de https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

📊 Análise Semântica de Detratores - E-commerce Olist


🎯 Objetivo do Projeto
Identificar as causas raízes de insatisfação dos clientes que atribuíram Nota 1 (Detratores), quantificar o impacto financeiro dessas falhas e mapear sua distribuição geográfica no Brasil.

🛠️ Tecnologias Utilizadas
Python (Pandas, Seaborn, Matplotlib)

NLP / Lógica de Keywords: Classificação semântica customizada para identificar motivos de reclamação.

Análise de Dados: Integração de múltiplos datasets (Reviews, Orders, Customers, Sellers).

📈 Principais Insights
Gargalo Logístico: A maior parte das reclamações está concentrada em problemas de Logística, especialmente nos estados de SP, RJ e MG.

Foco no Vendedor: Identificamos que falhas de Qualidade e Pedido Errado estão concentradas em um grupo específico de Sellers, permitindo ações diretas de auditoria.

Impacto Financeiro: O valor total de produtos sob risco de churn/devolução foi de R$1,129,861.81.
